# Maeve and Salvatore BioInfo Repository
## Details
- Group viromics project
- Bog frozen replicate B accession no. SAMN08784154
## Workflow Summary 
In this project, we analyzed viromics sequencing data from bog frozen replicate B. Our workflow included downloading raw FASTQ files, assessing read quality with FastQC, trimming low quality bases and adaptors with trimmomatic, checking cleaned reads with FastQC, assembling reads into contigs with MEGAHIT, and identifying viral contigs for downstream analysis. This repository contains notes, scripts and workflow documentation used to make the analysis reproducible. 
## Conclusion
Overall, this workflow allowed us to move from raw sequencing reads to assembled contigs and candidate viral sequences for further study. This project emphasized the importance of quality control, careful file organization, and reporducible scripting in bioinformatics workflows.
